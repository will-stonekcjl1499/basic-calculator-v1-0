## Overview

Basic Calculator is a web-based tool for everyday arithmetic tasks. It relies on HTML, CSS, and JavaScript, with DOM updates and event listeners driving the button-based interaction model in a clean, minimal interface.

This repository also works well as a small frontend learning sample. It is handy for studying calculator behavior, practicing browser interactivity, or using a simple base for adding more advanced calculator functions later.

---

## Features

- Performs addition, subtraction, multiplication, and division
- Provides an AC control to wipe the current entry
- Provides a DEL control to delete the most recent character
- Uses HTML for page structure, CSS for presentation, and JavaScript for interactivity
- Updates the screen through DOM manipulation
- Listens for calculator button events with event listeners
- Keeps the calculator logic on the frontend
- Works entirely in the browser and needs no extra runtime dependencies

## How to Use

1. Open the calculator in a web browser.
2. Type values using the on-screen buttons.
3. Pick an operator such as +, -, x, or /.
4. Press = to display the answer.
5. Use AC to start over from scratch.
6. Use DEL to remove the last input when you need to fix a value.

Example workflow:

- Click 12
- Select +
- Click 8
- Press =
- Result appears in the display
